Fonctions utiles en Python pur : map() et filter()

map(function, iterable) applique une fonction chaque lment dun itrable.

list(map(lambda x: x*2, [1, 2, 3]))  #  [2, 4, 6]

filter(function, iterable) garde les lments pour lesquels la fonction retourne True.

list(filter(lambda x: x % 2 == 0, [1, 2, 3, 4]))  #  [2, 4]

Ces outils sont trs utiles aussi dans Pandas (apply, map) !

Pandas La base

Importer pandas

import pandas as pd

Charger un fichier CSV

df = pd.read_csv('donnees.csv')

Afficher les premires lignes

df.head()

Structure du DataFrame

df.info()   # Infos gnrales
df.describe()  # Statistiques descriptives

Filtrer avec filter et conditions

Filtrer par colonne

df[['nom', 'ge']]

Filtrer par ligne (condition)

df[df['ge'] > 30]

Plusieurs conditions

df[(df['ge'] > 30) & (df['ville'] == 'Paris')]

Transformer avec map et apply

Crer une nouvelle colonne

df['anne_naissance'] = 2025 - df['ge']

Utiliser map() pour modifier une colonne

df['ville'] = df['ville'].map({'Paris': 'FR', 'Lyon': 'FR', 'Berlin': 'DE'})

Utiliser apply() pour appliquer une fonction

def categorie_age(age):
    if age < 18: return 'Mineur'
    elif age < 60: return 'Adulte'
    else: return 'Senior'

df['catgorie'] = df['ge'].apply(categorie_age)

Calculs statistiques simples

Statistiques de base

df['ge'].mean()     # Moyenne
df['ge'].median()   # Mdiane
df['ge'].std()      # cart-type
df['ge'].min(), df['ge'].max()

Grouper et agrger (groupby)

df.groupby('ville')['ge'].mean()

df.groupby('ville').agg({
    'ge': ['mean', 'median', 'count'],
    'salaire': 'mean'
})

Visualisation avec Seaborn

Importer seaborn et matplotlib

import seaborn as sns
import matplotlib.pyplot as plt

Histogramme

sns.histplot(df['ge'], bins=10)
plt.show()

Visualisation avec Seaborn

Boxplot

sns.boxplot(x='ville', y='ge', data=df)
plt.show()

Nuage de points

sns.scatterplot(x='ge', y='salaire', data=df)
plt.show()

Corrlation (heatmap)

corr = df.corr(numeric_only=True)
sns.heatmap(corr, annot=True)
plt.show()

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Charger les donnes
df = pd.read_csv('personnes.csv')

# Ajouter une colonne
df['anne_naissance'] = 2025 - df['ge']

# Filtrer
df_paris = df[df['ville'] == 'Paris']

# Statistique
print("ge moyen  Paris :", df_paris['ge'].mean())

# Visualisation
sns.histplot(df_paris['ge'], bins=10)
plt.title("Rpartition des ges  Paris")
plt.show()
